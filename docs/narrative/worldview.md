# 世界观设定

> 设定约束见 [rules.md](rules.md)。

## 1. 锈镇

镇子没有名字。管道和烟囱是它的骨骼，铁锈是它的皮肤。屠宰场从不熄灯。

你在这里待久了会发现，镇上的人已经不觉得铁腥味是味道了。它是空气的一部分，和潮湿一样理所当然。建筑从来没被重建过——只是修补，一层盖一层，像伤口上叠伤口。技术大约停在二十世纪中叶的某个时刻，然后就不再往前走了。机器还在转，但没人说得清是谁最初把它们装上去的。

镇子封闭。游戏全程不离开这个镇。不是围栏监狱式的封闭——路在，往外走，回过神来你还在原地。那种封闭更接近一个你醒不过来的梦。

社会由行会与帮会共治。屠夫行会控制经济命脉，酒馆是信息与地下交易的中枢。物种之间存在强弱的差异、阶层的落差，但不黑白分明。每个角色都有合理的动机，暴力与善意常常长在同一张脸上。没有人是彻底干净的。

## 2. 双世界

右眼看见的世界里，一切正常。酒馆在营业，市场在交易，工人在棚屋间进出，铁桥下的黑水缓慢流淌。人们说话，做事，维持着文明社会的运转。

切换到左眼。

同一面墙开始蠕动。同一个人的对话底下翻涌出另一层声音——不是他们在说谎，是他们在否认。否认自己最想藏住的恐惧，否认自己无法承认的欲望。影子世界不是另一座镇子，不是另一个维度。它是同一座镇子的精神层——每一块砖、每一张脸上都覆盖着被压抑之物的投影。环境变暗，扭曲，有机化。空气变重。

这是游戏规则，不需要故事去解释"为什么"。表面恒真。影子恒假。取影子之反，结合表面之真——这就是推理的全部。猫的异色瞳不是超能力，是侵入。他听见的不是谎言，是对方对自己撒的那个谎。

## 3. 超自然

没有魔法。没有怪物。没有神明。

这座镇子上的超自然，全部发生在精神层面。猫的异色瞳是其一——侵入性的读心，通过影子侧的对话触达对方的耻部。其二是精神畸变：少数人拥有不同程度的精神特质差异。有人能感知自己正被注视——被猫的左眼锁定时，皮肤发麻，本能想躲。有人能控制自己在影子侧的谎言暴露量——话说得更短，更碎，更会绕。其中一个人几乎完全光滑，猫的左眼抓不住任何东西。

精神畸变不是超能力体系。没有分级，没有名称，没有"修炼"。它更像有些人生来擅长说谎，有些人生来不擅长藏情绪——只是这道裂缝开得比常人大一些。

## 4. 场所

### 地面层

**后巷** `alley`

两侧都是建筑的背面。没有正脸的地方。湿砖墙，生锈管道，垃圾桶里的东西很久没人收了。猫是在这里醒来的。影子侧的墙在缓慢蠕动，管道里滴下来的液体不是水。

**市场** `market`

露天集市。摊位之间挤出来的过道只够一个人走，但永远有两个人并排站在里面吵架。叫卖声、赌博、传谣、偷窃——所有这些在表面侧是混乱的日常，在影子侧变成一种稠密的、胀满的贪婪。每个摊位都是一张脸压在玻璃上。

**酒馆** `tavern`

中立之地。吧台上的划痕比镇上最老的工人还老。所有人都在这里——打架的在角落，和解的在吧台，吹牛的在中间，交易的坐在最里面那桌，背对墙。老板什么都知道，什么都不说。影子侧的压抑是累积性的——每个人的不安堆在同一间屋子里，空气变得比外面更重。

**铁桥** `bridge`

跨过一条黑水运河。铁锈栏杆冰手。约会在这里，交易在这里，抛尸也在这里。桥下有东西在流水中移动——可能只是垃圾。影子侧的水面不是黑的，是某种在流动但不反射光的物质。

**旧诊所** `clinic`

藏在巷子深处的半废弃诊所。医生缝伤口，开药，不多问。你带什么伤来就治什么伤——不问是谁的刀。有些镇民受伤了不来这里，有些只来这里。影子侧的药瓶里，液体在无风自动。

**工棚区** `shacks`

密集的低矮住房。几户共用一个炉子。晾着的衣服永远干不透。大部分屠场工人住在这里——下班后穿着汗衫坐在门口，不说话，就坐着。影子侧每扇门后面都在传出重复的声音，像同一句话被不同的人反复说着。

### 工厂层

**屠宰场** `slaughterhouse`

镇的心脏。二十四小时运转。铁腥味从这里出发，覆盖每一条巷子。生产线、冷库、挂肉间。表面侧是一座高效运转的工厂——血腥，但有序。影子侧是裂缝最薄的地方：机器在呼吸，墙在出汗，流水线上的不只是肉。分三层——外围、产线区、禁区。越往深处走，裂缝越宽。

**骨场** `bone_yard`

屠宰场后面的废弃物堆积地。骨头，角，蹄，不要的部分。白森森一片，一望无际。风从这里经过时会打哨——骨头之间的缝隙把风声切成碎末。有人来捡东西卖。有人来这里做不想被看见的事。影子侧的骨头不是白的，是灰的。有些在动。

**炼脂车间** `rendering_plant`

屠宰场附属设施。废料在这里被炼成油脂和肥料。味道的穿透力极强——不只是臭味，是某种粘在鼻腔黏膜上的东西。工人多是年纪大的，肺已经熏坏了，换不了别的活。表面侧是高温蒸汽和轰鸣机器；影子侧空气是胶状的，每吸一口都像在吞固体。

**行会大厅** `guild_hall`

屠夫行会的正式场所。铁门，石墙，里头有暖炉。比镇上任何建筑都体面。决策在这里做出——真正的事不在这里发生。影子侧的墙上挂的不是装饰品，是某种契约留下的痕迹。旧的，干的，但没褪。

**废弃工厂** `abandoned_factory`

镇北一座停转多年的旧厂。机器还在，锈得厉害。成了临时住所、秘密会面点、避雨处。有人说里面不是空的。影子侧的机器偶尔发出声音——旧节奏，像在重复停转前最后一个动作，一遍，又一遍。

### 地下层

**排水道** `drains`

镇下的管网。宽处能站人，窄处只能爬。某些区域有住过的痕迹——毯子，空罐，烛泪。水流方向不太对，有时候倒着走。分上段和深处。能通到镇上不少地方的下面。影子侧的管壁在渗出东西——不是水，闻起来也不像任何该在管道里的东西。

**旧井** `old_well`

市场旁边一口盖着铁栅的深井。不供水了。往下看什么都看不见。但在影子侧往下看——深处有光。谁放的，什么时候放的，没人说得清。也许一直就有。

**镇边荒地** `outskirts`

镇子的边缘。不再是建筑——碎石，枯草，歪斜的铁丝网。再往外是走出去的方向，也是走不出去的方向。猫反复回到这里。影子侧和表面侧在此处几乎重叠——两个世界的界线最模糊，像醒和睡之间那个缝。
